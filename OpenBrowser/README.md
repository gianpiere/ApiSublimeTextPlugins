## Installation
Install this repository via [Package Control](http://jgianpiereramos.com,"Gianpiere")


## Usage
To insert or change a selected color, use:

- Linux: `ctrl+shift+b`
- Windows: `ctrl+shift+b`
- OS X: `cmd+shift+b`
