## Installation
Install this repository via [Package Control](https://github.com/gianpiere/ApiSublimeTextPlugins)


## Usage
To insert or change a selected color, use:

- Linux: `ctrl+shift+b`
- Windows: `ctrl+shift+b`
- OS X: `cmd+shift+b`
